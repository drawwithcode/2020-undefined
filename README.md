# A Connected Garden
A Connected Garden is a project built within the course [Creative Coding](https://drawwithcode.github.io/2020/) at the Politecnico di Milano. It makes use of the [p5.js](https://p5js.org) library and and [Mappa.js](https://mappa.js.org). In addition it stores the data on [firebase](https://mappa.js.org).

# How to Run
Be sure to have node installed: https://nodejs.org/
* Install node dependencies: `npm install`
* Run local server: `node app.js`
* Open: `localhost:3000`
